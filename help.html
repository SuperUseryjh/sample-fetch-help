OICPP sampleTester - 使用指南

欢迎使用“OICPP sampleTester”油猴脚本！本脚本旨在帮助您从支持的在线评测系统（OJ）页面中快速提取代码样例，并将其发送到本地的 OICPP 以创建新的题目。

重要提示： 本脚本需要您已安装并运行 OICPP 以及 Tampermonkey 浏览器扩展。

一、前置条件

  1. 安装 Tampermonkey 扩展：

    请确保您的浏览器（Chrome, Edge, Firefox 等）已安装 Tampermonkey 扩展。您可以从各自的浏览器应用商店安装它：

    • Chrome 应用商店
    • Edge 附加组件
    • Firefox 附加组件
  2. 运行 OICPP 服务：

    本脚本需要一个在本地运行的本地 OICPP 服务来接收数据。请确保您的 OICPP 服务已启动并正常运行。

二、安装脚本

  1. 获取脚本文件：

    您应该有一个名为 fetch_tampermonkey.user.js 的油猴脚本文件。

  2. 导入到 Tampermonkey：
    • 打开您的浏览器，点击 Tampermonkey 扩展图标。
    • 选择“创建新脚本”或“仪表板”。
    • 如果您选择了“创建新脚本”，请删除所有默认内容，然后将 fetch_tampermonkey.user.js 文件的全部内容复制并粘贴到编辑器中。
    • 如果您进入了“仪表板”，可以点击左侧的“实用工具”选项卡，然后找到“从文件导入”或“从 URL 安装”选项。选择您的 .user.js 文件进行导入。
    • 保存脚本（通常是点击文件菜单中的“保存”图标或按下 Ctrl + S）。
  3. 确认脚本已启用：

    在 Tampermonkey 仪表板中，确保“样例抓取器”脚本旁边的开关已打开（绿色或蓝色），表示它已启用。

三、使用脚本

脚本会在以下支持的 OJ 网站上自动激活：

  • https://luogu.com.cn/*
  • https://www.luogu.com.cn/*
  • https://htoj.com.cn/*
  • https://atcoder.jp/*
  • https://codeforces.com/*

操作步骤：

  1. 访问题目页面：

    在上述任何一个支持的 OJ 网站上，打开一个题目页面。

  2. 寻找“样例抓取器”按钮：

    页面加载后，您会在浏览器窗口的右上角看到一个名为“⬇️”的蓝色按钮。这是用于下载样例的按钮，其可以被拖动。

  3. 点击按钮：

    点击此按钮。脚本将尝试自动提取页面上的样例代码和题目名称。

    • 如果成功提取，它将把数据发送到您的本地 OICPP 服务，并在按钮附近显示一个临时的状态消息（例如“成功: 题目已创建”）。
    • 如果遇到问题（例如未找到样例或 OICPP 服务未运行），会弹出警告框并显示错误消息。
  4. 手动面板（仅限 HTOJ 以外的网站）：

    在某些网站上，点击按钮可能会弹出一个更详细的面板，允许您手动输入 OJ 名称和题目名称，然后点击“提取代码并创建题目”按钮。

四、故障排除

  • 脚本未显示或未工作：
    • 确保 Tampermonkey 扩展已安装并启用。
    • 在 Tampermonkey 仪表板中，检查“样例抓取器”脚本是否已启用。
    • 刷新题目页面。
    • 检查浏览器控制台（按 F12 打开）是否有任何错误消息。
  • “请求失败”错误：
    • 这通常意味着 OICPP 服务未在 http://127.0.0.1:20030 运行或无法访问。请确保您的 OICPP 服务已启动。
    • 检查您的防火墙设置，确保它没有阻止浏览器与本地 OICPP 服务之间的通信。
  • 未找到任何 标签可提取：
    • 这表示脚本无法在当前页面上找到预期的代码样例元素。这可能是因为网站结构发生了变化，或者当前页面没有可识别的样例。
    • 您可以尝试在 Tampermonkey 脚本中修改 DOMAIN_CONFIG 部分的 codeSelectors 来适应新的网站结构（如果您了解 JavaScript 和 CSS 选择器）。
    • 如果您没有此类经验，可以联系开发人员并让他们进行修改

如果您遇到其他问题，请联系脚本作者或查阅 OICPP 的相关文档。
